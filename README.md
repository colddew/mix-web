=======

- basic framework: springmvc、spring、mybatis
- templating engine: freemark、velocity
- security: shiro、oauth
- workflow: jbpm
- rules engine: drools
- rdbms: mysql

- cache: memcached
- rpc: dubbo

- deploy: maven、jetty、tengine
